# Kalkulator Internetowy

Prosty kalkulator stworzony przy użyciu **HTML, CSS oraz JavaScript**, pozwalający na wykonywanie podstawowych operacji matematycznych bezpośrednio w przeglądarce.

## Główne cechy
* **Podstawowe obliczenia:** dodawanie, odejmowanie, mnożenie i dzielenie.
* **Responsywność:** aplikacja działa płynnie na różnych urządzeniach.
* **Interaktywność:** obsługa zdarzeń w czasie rzeczywistym dzięki JavaScript.

## Cel projektu
Projekt został zrealizowany w celu nauki logiki programowania oraz tworzenia dynamicznych interfejsów użytkownika.
