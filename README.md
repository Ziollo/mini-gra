# Mini Gra Webowa

Projekt prostej gry zręcznościowej stworzonej w celu przećwiczenia zaawansowanych aspektów języka JavaScript oraz animacji CSS.

## Funkcje projektu
* **Dynamiczna rozgrywka:** Mechanika oparta na zdarzeniach (event listeners) i funkcjach czasu.
* **System punktacji:** Śledzenie wyników gracza w czasie rzeczywistym.
* **Responsywny design:** Gra dostosowana do różnych rozdzielczości ekranu.

## Technologie
* **HTML5** (Struktura gry)
* **CSS3** (Stylizacja i animacje)
* **JavaScript (ES6)** (Logika gry i interakcje)

## Czego się nauczyłem?
Podczas tworzenia tej gry zgłębiłem wiedzę z zakresu zarządzania stanem aplikacji, optymalizacji renderowania elementów oraz tworzenia prostych algorytmów sterujących zachowaniem obiektów na ekranie.
